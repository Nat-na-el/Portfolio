Projects

Forex Trading Bot

Developed a Python-based automated Forex trading bot using technical indicators such as Moving Averages, RSI, and Bollinger Bands. The bot runs backtests on historical market data to optimize entry and exit strategies, aiming to maximize profit while controlling risk. Designed for modularity, it allows easy integration of new trading algorithms and supports live trading environments.

Backend APIs

Created RESTful APIs using Python and Flask with a focus on scalability, security, and performance. Features include JWT authentication, role-based access control, PostgreSQL database integration, and comprehensive unit and integration testing. These APIs serve as the backbone for various applications requiring reliable and secure data services.

Financial Data Dashboard

Built an interactive dashboard using Flask and Chart.js that visualizes Forex market data in real-time. Users can track price movements, trading volume, volatility, and sentiment indicators with customizable views. The dashboard supports live data streaming and provides intuitive charts to assist traders in making informed decisions.
